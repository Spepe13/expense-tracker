# 💰 Διαχείριση Οικονομικών

Εφαρμογή για παρακολούθηση δόσεων και εξόδων.

## 🚀 Ανέβασμα στο Vercel (5 λεπτά)

### Βήμα 1: Ανέβασε στο GitHub

1. Πήγαινε στο [github.com](https://github.com) και κάνε login
2. Κλικ στο **"+"** πάνω δεξιά → **"New repository"**
3. Δώσε όνομα: `expense-tracker`
4. Κλικ **"Create repository"**
5. Ανέβασε τα αρχεία:
   - Κλικ **"uploading an existing file"**
   - Σύρε όλα τα αρχεία του φακέλου μέσα
   - Κλικ **"Commit changes"**

### Βήμα 2: Σύνδεσε με Vercel

1. Πήγαινε στο [vercel.com](https://vercel.com)
2. Κλικ **"Sign Up"** → **"Continue with GitHub"**
3. Κλικ **"Add New Project"**
4. Βρες το `expense-tracker` και κλικ **"Import"**
5. Κλικ **"Deploy"**
6. Περίμενε 1-2 λεπτά...
7. ✅ Έτοιμο! Παίρνεις URL τύπου: `expense-tracker-xxx.vercel.app`

### Βήμα 3: Πρόσθεσε στο iPhone

1. Άνοιξε το Safari στο iPhone
2. Πήγαινε στο URL που πήρες (π.χ. `expense-tracker-xxx.vercel.app`)
3. Πάτα το κουμπί **Share** (το τετράγωνο με το βέλος)
4. Πάτα **"Add to Home Screen"** / **"Προσθήκη στην αρχική οθόνη"**
5. Πάτα **"Add"** / **"Προσθήκη"**

Τώρα έχεις την εφαρμογή σαν app στο iPhone! 📱

## 📝 Χρήση

### Δόσεις
- Πρόσθεσε προγράμματα δόσεων (δάνεια, κάρτες κλπ)
- Τσέκαρε κάθε δόση όταν την πληρώνεις
- Βλέπεις πρόοδο και υπόλοιπο

### Έξοδα
- Πρόσθεσε έξοδα με κατηγορίες
- Σημείωσε όταν πληρώνεις
- Φιλτράρισμα πληρωμένων/απλήρωτων

### Backup
- **Εξαγωγή**: Κατέβασε JSON με όλα τα δεδομένα
- **Εισαγωγή**: Φόρτωσε δεδομένα από backup

---

## ⚠️ Σημείωση για Sync

Τα δεδομένα αποθηκεύονται **τοπικά** σε κάθε συσκευή (localStorage).
Αν θες τα δεδομένα να είναι συγχρονισμένα μεταξύ Mac και iPhone, 
χρησιμοποίησε τα κουμπιά Εξαγωγή/Εισαγωγή ή ζήτα μου να προσθέσω 
cloud sync με Supabase.
